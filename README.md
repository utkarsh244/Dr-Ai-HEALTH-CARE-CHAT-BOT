# 🩺 Dr.AI - Medical AI Assistant (Frontend)

A modern React.js frontend for an AI-powered Medical Assistant that helps users interact with an intelligent healthcare chatbot. The application provides a clean, responsive interface for medical conversations, report generation, emergency support, and document management through an Admin Panel.



## 🚀 Features

- 🤖 AI-powered Medical Chat Assistant
- 💬 Interactive Chat Interface
- 📄 PDF Report Generation
- 🩺 Medical Knowledge Base Integration (RAG)
- 📁 Admin Panel for PDF Upload & Management
- 🗑️ Document Delete Functionality
- 📊 RAG Status Monitoring
- 🌙 Dark / Light Theme Support
- 🆘 Emergency SOS Button
- 🗺️ Leaflet Map Integration
- 🔥 Firebase Authentication & Storage
- 📧 EmailJS Integration
- 📱 Fully Responsive UI


### Libraries

- Axios
- Firebase
- EmailJS
- jsPDF
- Leaflet
- React Scripts
